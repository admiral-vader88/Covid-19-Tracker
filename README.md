# Covid-19-Tracker
COVID-19 Tracker using Tkinter
This is a simple Python application that displays the total active and confirmed COVID-19 cases in India using data from the Covid19India API. The application uses the Tkinter library to create a graphical user interface (GUI) and the requests library to fetch data from the API.

Features
Displays total active COVID-19 cases in India.
Displays total confirmed COVID-19 cases in India.
Provides a "Refresh" button to update the data in real-time.
Requirements
Python 3.x
requests library
tkinter library (usually included with Python installations)
